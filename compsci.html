JAVA



Matrix Word Finder
Backtracking

This Java Project utilized Backtracking and Recursion to create a word finder. Given an MxN matrix of characters and a specific word to find, the program searches the matrix for the word. If the word is found, the program will visually display the word in the aforementioned matrix, and if the word is not found, it will be reported as such.


Words can be found horizontally, vertically, diagonally, and with a combination of all three. This makes backtracking through the matrix a difficult task, but can be done with precise navigation through a 2D Array of characters.









Maze Magic
Breadth-First-Search

This Java Project utilized Breadth-First-Search (BFS) to find the lowest number of moves a person can make to escape a given maze (MxN matrix). Each square of the maze has its own properties, such as forbidden squares that cannot be accessed (!) teleportation squares (any letter, such as A to A or B to B), and exit squares ($).


Using BFS, as well as techniques such as a custom Graph class and a Hashmap with an ArrayList of coordinates for each key, this program was able to tackle massive test cases (1000x1000) in mere seconds.








 

C/C++



Arcade Manager
Stacks & Linked Lists

This C Project utilized Linked Lists and Stacks to manage the players that come into an arcade, as well as their tokens. Input is taken from the user: if the input is 1, then a new player is entering the arcade and the user must give a string (name) and integer (tokens). If the input is 2, the player who entered the arcade most recently leaves (a Stack utilizes First In Last Out). If the input is 3, the program will output the name of the player who entered with the most tokens that is still currently in the arcade.


The strategy of this program involved using a Linked List of nodes that store a "Player" pointer to the player currently in the node, a "Player" pointer to the current MVP (player with the most tokens), and a "Node" pointer to the next node. I also used a Stack to perform FILO insertion/removal of players, as described by the assignment instructions.











Scientific Calculator
Dynamic Memory, File I/O, Etc.

This C Project is an interactive scientific calculator with all functionality that one would expect in a scienfitic calculator. Trigonometric functions such as sine, cosine, tangent, and their inverses are included as well as basic functions such as multiplication and exponentation. Numbers can be stored and recalled in variables with the "sto" and "rcl" commands, and users can access previous calculations via the "b" command, all of which required the use of dynamic memory allocation.


Command history is outputted to an external tape file when the program concludes. This was the final project for my first ever programming course, and I am still proud of it to this day.
